【转载】木格子窗户

发表于 2018-07-28 | 分类于 读书/作文 | | 阅读次数:

注：这篇散文是高三时偶然从Jungle的杂志《青春美文》中读到，作者”麦田里的乌鸦”，原文如今在网络中是踪迹难寻了，特保存于此。

那是一扇木格子窗户，温润的咖啡色，手指轻轻抚过，有凹陷下去的柔软。阳光直直地透过菱形的玻璃，把屋里照出一块明黄色的透明。灰尘飞舞是青草飞花的味道，闭上眼睛，你可以闻到稀薄流动的芬芳。

喜欢这扇窗户，它已经陈旧得没了傲气，多出来轻轻松松的安详，淡然得像缓缓的旋律，只是流淌。

没事的时候，喜欢对着它说话，很长很长的故事，大段大段琐碎的独白。

有时候想，我的话会不会变成它流畅的纹理，变成它温润颜色里的一点点深。它晒着太阳或者碰触到雨水的时候，会不会想起我的声音，年轻的女孩子寂寞而骄傲的声音，很慢很慢的倾诉，很多很多的独白。

时光在某一刻是可以凝固住幸福的，我一直相信，坚定不移地相信着那迅疾吹过心上的时间的风。它一点一点把我变得美好起来，它用微小的羽毛替我挡下尖锐而锋利的穿行，它用平静的脸替我回答那些固执而血淋淋的问题，它用绵软的手替我抚平那些起伏的忐忑和悲喜。

我想我应该感恩。我们彼此远远地观望，我不停地想抛弃它轰轰烈烈地独自上路，它却对我不离不弃，为我挡下伤害。

我再一次站在这扇窗户面前。它的咖啡色上有黑色水笔的清浅字迹，是很多年前男孩子默默写下的祝福，凌乱而苍白。我可以想象他的手指或许有微微的颤抖，他的脸上或许映照着窗户淡淡的影子，他的头发柔顺地散在眼睛的周际，他的心里或许在重复一个名字，难以抑制。

我永远无法知道那圆润的字句是他写给谁的心情。那些经年后陈旧的一切，曾经怎样微笑地看着一个如树叶般蓬勃的男孩子带着清澈的笑容慢慢描述出一段温暖的话。木格子窗户，有一颗彩色的木质的心，它留下了痕迹，却遗忘了思绪，任那个男孩晃悠着飘出自己的视线。它用很长很长的岁月把羞涩的美好演绎成云淡风轻的过往。那些粉红色的暖昧也成了安定下来的回忆，寂静地陪着它看沧海，看桑田，看外面的世界离离合合。

有一瞬间，我想起了玛丽・黑。那个声音平铺直叙的爱尔兰女子，平和地表现音乐，歌声却蕴涵着最为广阔的自由，在风里穿行没有束缚。我也想起你暗哑的声音，还有你额前的头发随着你的手上下翻飞，你那么欢欣地讲述这个女子的故事。你读出那些温婉的词语，你讲述那些直指人心的幸福故事，你的身边有水淌过一样的安静，我可以听到外面的树叶飞扬起来的笑声。

我轻轻摩挲那些字迹，好像在感触自己的心。我用指尖在上面留下浅浅的曲线，是游离在木色纹理里的寂寞的鱼。有没有人能看懂呢?会不会看着它们就想起自己水一样的轻愁?还是很多年以后，那些淡淡的线条也长成了流畅的纹理，融在了这扇窗户上，成了它的记忆?

我不得而知。我的心安静着，不想说话。

终有一天，这扇木格子窗户要被岁月抹掉。

它的咖啡色会褪成更为干净的草白，它的菱形玻璃会有花纹一样繁复好看的痕迹，甚至通透了外面的喧嚣。它平淡的心或许会离开它的身体，飞舞在终年有太阳有笑声的地方，把它那么长时间以来留下的记忆好好地回味，发出孩子―样明朗的笑声。

我微笑，我知道，总有一天，我对它说的所有的故事都会跟着它离开，陪着它快乐地行走，就像当初它陪着我一样。
